使用说明
--------
1. 这些代码为聚宽平台（joinquant.com）策略代码，需在聚宽平台上运行。
2. 直接克隆到聚宽研究环境即可，无需额外安装依赖（聚宽已内置jqdata/jqfactor等库）。
3. 如需迁移到其他平台，请参考配套PDF报告《聚宽量化策略精选深度评估报告》中
   每个策略的"迁移建议"章节。
4. 部分策略依赖jqfactor专有因子库（如策略#5的TVSTD20、策略#7的sales_growth、
   策略#9的多因子库），迁移时需自行实现这些因子。
